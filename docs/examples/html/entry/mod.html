image2D.js 是一个专注于前端二维绘图的JavaScript库，本项目主要是用于演示一些基于此开发的用例，欢迎你的加入。

我们推荐你使用npm方式引入： 

import $$ from 'image2d';

当然，你也可以直接通过CDN引入（请选择你使用的版本）： 

<script src='https://cdn.jsdelivr.net/npm/image2d@1.12.4/build/image2D.min.js'></script>

在使用的过程中，你可以查看 更新日志 来判断是否需要升级或应该选择那个版本。当然，我们提供向下兼容，因此在大部分情况下，最新版本可能都是比较好的选择！
